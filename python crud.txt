# 🔹 CRUD on a Python Dictionary

CRUD = **Create, Read, Update, Delete**

We’ll use this example dictionary:

```python
contacts = {
    "Alice": "123456",
    "Bob": "987654"
}
```

---

### ✅ 1. **Create** (add a new key–value pair)

```python
contacts["Charlie"] = "555555"
print(contacts)
# {"Alice": "123456", "Bob": "987654", "Charlie": "555555"}
```

---

### ✅ 2. **Read** (access a value by key)

```python
print(contacts["Alice"])      # 123456
```

⚠️ If the key doesn’t exist → `KeyError`.
Better way:

```python
print(contacts.get("Alice"))        # 123456
print(contacts.get("Daniel", "Not found"))  
# "Not found" because Daniel isn’t in the dictionary
```

---

### ✅ 3. **Update** (change a value)

```python
contacts["Alice"] = "000000"   # overwrite
print(contacts["Alice"])       # 000000
```

---

### ✅ 4. **Delete**

```python
# Method 1: del
del contacts["Bob"]
print(contacts)   # {"Alice": "000000", "Charlie": "555555"}

# Method 2: pop (returns the deleted value)
removed = contacts.pop("Charlie")
print(removed)    # 555555
print(contacts)   # {"Alice": "000000"}
```

---

# 🔹 Other very useful dictionary methods

### 🔸 `.keys()` → get all keys

```python
print(contacts.keys())   # dict_keys(['Alice'])
```

### 🔸 `.values()` → get all values

```python
print(contacts.values())  # dict_values(['000000'])
```

### 🔸 `.items()` → get key–value pairs

```python
for name, number in contacts.items():
    print(f"{name} → {number}")
```

---

### 🔸 `.update()` → add or update multiple entries

```python
contacts.update({"Bob": "222222", "Eve": "333333"})
print(contacts)
# {"Alice": "000000", "Bob": "222222", "Eve": "333333"}
```

---

### 🔸 `.clear()` → remove everything

```python
contacts.clear()
print(contacts)   # {}
```

---

# ✅ Summary

* **Add / Update:** `contacts["Name"] = "Number"`
* **Read safely:** `contacts.get("Name", "Not found")`
* **Delete:** `del contacts["Name"]` or `contacts.pop("Name")`
* **Loop through:** `.keys()`, `.values()`, `.items()`
